# ERC3 Benchmark Specifications

This directory contains exported specifications for all ERC3 benchmark variants.

- `erc3-dev_specs.md` - Development benchmark (16 tasks)
- `erc3-test_specs.md` - Test benchmark (24 tasks)  
- `erc3-prod_specs.md` - Production benchmark (103 tasks)

Each file includes task descriptions, gotchas, and available API routes for that benchmark variant.
